# Python Process. Инференс модели YOLOv8 Pose


Данная программа выполняет обработку видео с помощью модели `yolov8s-pose` из библиотеки `ultralytics`.

Цель работы — ускорить инференс модели на CPU за счет распараллеливания обработки кадров видео с использованием процессов Python.

Программа:
- считывает видеофайл;
- разбивает кадры на части;
- обрабатывает кадры моделью `YOLOv8s-pose`;
- наносит KeyPoints на каждый кадр;
- собирает обработанные кадры обратно в видео;
- измеряет время обработки.
---
Пример ввода:
```bash
python task5/main.py --path task5/example_in.MP4 --mode 1 --name example_out
```
Пример вывода:
```bash
Видео task5/example_in.MP4 обработано за 40.06175441600499с на 1 процессах и сохранено под названием example_out.mp4
```
