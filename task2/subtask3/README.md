### Параллельная реализация решения системы линейных алгебраических уравнений с помощью OpenMP

### Где:
* Вариант 1: для каждого распараллеливаемого цикла создается
отдельная параллельная секция #pragma omp parallel for.
* Вариант 2: создается одна параллельная секция #pragma omp
parallel, охватывающая весь итерационный алгоритм.

Главная диагональ матрицы А = 2.0, остальные элементы = 1.0. 
Все элементы вектора b равны N+1. 
В этом случае решением будет вектор, элементы которого = 1.0


### Сборка варианта 1

```bash
rm -rf build
mkdir build && cd build
cmake ..
make
./slau_v1
```
