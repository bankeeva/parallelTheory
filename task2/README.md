# OpenMP Labs

В репозитории представлены три задания и отчеты по ним:

- `subtask1/` — умножение матрицы на вектор
- `subtask2/` — численное интегрирование
- `subtask3/` — решение СЛАУ методом простой итерации

Каждая папка содержит:
- исходный код программы,
- результаты измерений,
- графики,
- отдельный `README.md`.
