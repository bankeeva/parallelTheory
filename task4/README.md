# Python Thread — лабораторная работа

Программа считывает данные с камеры и трёх датчиков `SensorX` в отдельных потоках, после чего отображает итоговое изображение в окне OpenCV.

- `SensorCam` считывает кадры с камеры
- `SensorX` имитируют датчики с частотой:
  - 100 Hz
  - 10 Hz
  - 1 Hz

Главный поток получает последние данные от всех датчиков через очереди `queue.Queue`, накладывает значения датчиков на кадр с камеры и отображает результат в окне.

При нажатии клавиши `q` программа корректно завершает работу и освобождает ресурсы.
