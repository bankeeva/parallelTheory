# Решение уравнения теплопроводности (2D) с использованием OpenACC

Реализация численного решения **уравнения теплопроводности** в двумерной области с помощью **явной разностной схемы** (пятиточечный шаблон Якоби).


- **Размеры сеток**: `128×128`, `256×256`, `512×512`, `1024×1024`
- **Граничные условия**: линейная интерполяция по краям между угловыми значениями
- **Значения в углах**:
  - `(0,0)` → 10
  - `(N-1,0)` → 20
  - `(0,N-1)` → 20
  - `(N-1,N-1)` → 30
- **Внутренняя область** изначально заполнена нулями
- **Тип данных**: `double`
- **Критерии остановки**:
  - Максимальная ошибка < `1e-6`
  - Максимальное число итераций = `1 000 000`
