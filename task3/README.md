# Task 3 — Многопоточное программирование на C++ с использованием `<thread>`

В репозитории представлены два задания:

- `subtask1/` — многопоточное умножение матрицы на вектор
- `subtask2/` — клиент-серверное приложение с очередью задач

Каждая папка содержит:
- исходный код программы,
- результаты работы,
- необходимые входные/выходные файлы,
- отдельный `README.md` с описанием реализации и инструкцией по запуску.
