## Многопоточное клиент-серверное приложение

Сервер:
- получает задачи из общей очереди;
- выполняет вычисления в рабочих потоках;
- сохраняет результаты в контейнер результатов.

Клиенты:
- работают в отдельных потоках;
- отправляют задачи на сервер;
- ожидают результаты вычислений;
- записывают результаты в файл.

В текущей реализации создано 3 клиента:
- клиент вычисления квадратного корня;
- клиент вычисления синуса;
- клиент возведения числа в степень.

### Сборка

```bash
rm -rf build
mkdir build && cd build
cmake ..
make
./srevcl
```

Результаты работы:
- Основные результаты вычислений сохраняются в файл results.txt.
- Результаты проверки сохраняются в файл test_results.txt.
